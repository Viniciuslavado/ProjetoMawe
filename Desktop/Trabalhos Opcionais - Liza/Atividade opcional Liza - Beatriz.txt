Dupla de feedback Beatriz Rodrigues 	

1- Usar uma programação que todos funcionários tenham acesso (gira) para verificar como está o processo de atualização
e deixar todos do cargo acima cientes do processo.

2- Usar um meio de comunicação geral, já detectado todos os problemas, erros e reparos que tem que ser feitos para
resolve-los.

3- Erro de comunicação, a equipe poderia sempre contatar para informar como está sendo produzido o website, fazer
demonstrações de como está ficando, ou também pode mostrar um esboço no início do projeto para verificar se 
deseja alguma alteração.

4- A equipe pode realizar uma reunião por video com ela, para ela explicar o que estava fazendo dentro do projeto 
para continuar sem a presença dela, realizando feedbacks para alinhar questões pendentes.
